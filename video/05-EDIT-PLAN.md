## 3. FINISHING / GRADE

**Per-portal grade direction (consistency anchored to brand cream + black):**
- **Cold open (Act 1‚Äì2):** crush blacks toward `#0e0a06`, warm midtones, persimmon `#ef6a3a` as the only saturated hue. Low-key, slight teal-less warmth.
- **Home:** neutral, clean, paper-cream `#E9E2D0` lifted whites; black type at full contrast; red `#e60000` accent only on accent words/CTAs.
- **Student (red):** warm up 100‚Äì150K, gentle red accent push on UI chrome; do NOT recolor real UI screenshots beyond a global LUT ‚Äî preserve true app colors.
- **Kitchen (tomato `#d52821`):** slightly hotter than student, marginally higher contrast for "operational" feel.
- **Admin (chartreuse/lime):** cool the field with `#cdfa50`/`#8fb305` accents on graphics/lower-thirds; keep dashboard screenshot true-color (data must read honest).
- **Close:** return exactly to home grade.

**Grain / vignette:**
- Cartoon scenes: very fine film grain 2‚Äì3% + soft 8‚Äì10% vignette to add "premium" depth.
- Real recordings: **no grain, no vignette** on the UI itself (keep crisp/legible); apply only to the framing matte/background behind a phone or browser mockup.
- Consistency rules: one master LUT + per-act offset; same vignette radius across cartoon scenes; never let grade reduce UI text legibility; keep brand red exactly `#e60000` everywhere (do not let per-portal grade drift the red).

**Audio loudness:** master to **-14 LUFS integrated**, true peak ‚â§ -1 dBTP. Music bed -18 to -16 LUFS ducked under any VO; SFX peaks ‚â§ -6 dBTP. One beat-drop alignment at 0:31 (0%). If no VO, music carries ‚Äî ensure the cut still reads on mute regardless.

**Captions (always-on, burned-in):**
- Style: bold sans (Inter/Archivo Bold) or the condensed grotesque used for titles, white fill, 2‚Äì3px black outline + 40% drop shadow for contrast on any background.
- Position: lower third, safe-margin 8% from bottom, centered; never overlap lower-thirds or the OTP/KPI punch-ins (raise caption or move punch-in up when they collide).
- Size: ~4.5% of frame height (1080p ‚âà 48px), max 2 lines.
- Burned-in for the muted-first watch. Sync to ‚â§1 frame.
